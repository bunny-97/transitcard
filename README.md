# 🚇 서울권 학생 맞춤 교통비 계산기 🚍  

대학교 생활하며 대중교통을 자주 이용하다 보니 교통비 부담이 커졌죠?  
특히 K-Pass, 기후동행카드 등 다양한 교통카드가 있지만,  
어떤 카드가 내게 가장 유리한지 직접 계산하고 비교하기는 너무 번거롭고 복잡해요😓  

이 프로그램은 여러분의 **나이와 교통수단(지하철, 버스, 마을버스, 직접입력)** 정보를 바탕으로  
1회 교통요금을 똑똑하게 계산해주고,  
주간 이용 횟수를 입력하면 월간 교통비도 자동으로 추정해줍니다📊  

복잡한 카드별 할인 정책도 자동으로 비교해주니,  
시간과 수고를 아끼면서 내게 딱 맞는 교통카드를 쉽게 찾을 수 있어요!  

서울권 학생을 포함해 교통비 절감에 관심 있는 분들에게 실질적인 도움이 될 거예요✨  

▶ 웹에서 바로 체험해보세요!  
[https://mytransitcard.netlify.app/](https://mytransitcard.netlify.app/)

### 주요 기능  
- 사용자 정보 기반 1회 교통요금 계산  
- 주간 이용 횟수 → 월간 예상 이용 횟수 및 총 교통비 자동 산출  
- K-Pass, 기후동행카드 등 카드별 복잡한 할인 정책 자동 비교  
- C++ 클래스 객체지향 설계 (User, RecommendationEngine)  
- STL 활용 (std::string, std::min, static_cast 등)  
- extern "C"로 JavaScript와도 연동 지원
